<a id="user-content-one-chip-to-rule-them-all" class="anchor" href="#one-chip-to-rule-them-all" aria-hidden="true"><svg class="octicon octicon-link" viewbox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>One Chip to Rule them All</h2>
<p>For discussion, find us at <a href="http://www.atari-forum.com/viewforum.php?f=115" rel="nofollow">Atari-Forum</a>.</p>
<p>Get your own MIST board e.g. from <a href="https://lotharek.pl/productdetail.php?id=45" rel="nofollow">Lotharek in Poland</a>, <a href="https://www.dragonbox.de/de/364-mist-fpga-v13-plus-midi-konsolen-4260416650978.html" rel="nofollow">Dragonbox shop in Germany</a>, <a href="https://www.amedia-computer.com/en/accueil/275-mist-midi-13-board-black-metal-case.html" rel="nofollow">AMedia in France</a>, or <a href="https://amigastore.eu/en/358-mist-midi-fpga-computer-with-midi-add-on.html" rel="nofollow">Amigastore in Spain</a>!</p>
<p><img src="https://raw.githubusercontent.com/wiki/mist-devel/mist-board/mist-midi-13-board-black-metal-case.jpg" alt=""></p>
<p>The MIST board was designed to implement classic 16 bit computers like the Amiga, Atari ST(E) or the Apple Macintosh (and even early 32 bit computers like the Acorn Archimedes) as a System-on-a-Chip using modern hardware. But it equally well supports 8 bit systems like the Atari XL, ZX81, ZX Spectrum, C64, Atari VCS, Atari 5200, Colecovision, Apple II, Sega Master System, Nintendo Gameboy, Nintendo NES, Odyssey2 and many many more ...</p>
<p>The MIST board is not about emulation. It's about the <a href="wiki/WhatIsThis">re-implementation of the original hardware itself in so-called FPGAs</a>.</p>
